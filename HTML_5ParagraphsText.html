What is a Paragraph?

In school, you learned a paragraph is a 3-5 sentence block of text that develops one main idea followed by supporting ideas or sentences.

In HTML, however, a paragraph is used to group text together on a webpage.

Paragraphs always start on a new line, and browsers automatically add whitespace before and after an HTML paragraph for organization and readability.

When Should You Use Paragraphs?

Paragraphs should be used ONLY for normal text content such as descriptions, explanations, stories, or ideas

UNLIKE HEADINGS, paragraphs are NOT to be used for titles or section names. Instead they are used to display the main witten content of a webpage

How to Create Paragraphs?

You create paragraphs using the <p> tag

EXAMPLE:

<p>This is a paragraph</p>

OUTPUT:

This is a paragraph

Ways to Format Paragraphs

You can style words inside a <p> tag

Types of formating:

  • <strong> – makes important text bold
  • <b> – makes text bold (visual only)
  • <em> – makes text italic and adds emphasis
  • <i> – makes text italic (visual only)

These tags are often used inside paragraphs to highlight important words or ideas

When to Use Basic Text Formatting

As mentioned earlier, you want to emphasize or highlight certain words inside a paragraph.

The purpose of text formatting is to draw important information without changing the structure of the page

Bold vs Strong

<strong> and <b> both make text bold, but they are used differently.

  • <strong> is used for important or meaningful text (recommended)
  • <b> is used only for visual bold styling

Italic vs em

<i> and <em> both make text italic, both make text italic. Also, same rules as above apply to italic text:

  • <em> is used to emphasize important text (recommended)
  • <i> is used for visual italic styling only

------

When To Use Bold and Italics

As mentioned, use <strong> and <em> most of the time

However using bold and italics do have some pupose.

Use BOLD For:

  • Keywords in a paragraph
  • Labels
  • UI text(like buttons or headings in simple designs with CSS)
  • Product Names

EXAMPLE: Make sure to click the Submit button when you finish an online quiz

------

Use ITALICS For:

  • Titles (Movies, Books, Songs)
  • Foreign Words
  • technical Terms

OVERALL:

  • <strong> and <em> = meaning
  • <b> and <i> = visual styling

EXAMPLE: The word annyeong is Korean
