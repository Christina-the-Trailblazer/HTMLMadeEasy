What are Links?

Links allow users to click from one page to another

You can also use links to jump to another document or another section of your page

The href Attribute and the <a> Tag

The <a> tag defines and creates a hyperlink that will eventually connect one source to another

The href attribute specifies the destination of the link

Example

<a href="url">Text Describing Link - Link title</a>

How to Create Different Links

Making a Basic Link

Code

<a href="https://www.google.com">Link to Google.com</a>

Output

Link to Google.com

Contact Information

For Emails:

Use mailto: inside href

<a href="mailto:example@email.com">Email: example@email.com</a>
example@email.com

For Phone Numbers:

Use tel: inside href 

<a href="tel:+15551234567">+1 (555)123-4567</a>
+1 (555)123-4567

For Addresses:

Just look for the address on Google or Apple maps and add it inside href In this example, I will do the Grand Canyon in Arizona 
<a href="https://maps.google.com/?q=Grand+Canyon+Arizona+86052">Grand Canyon</a>
Grand Canyon

Same Page Navigation

Did you know you can also jump to a specific section on the same page without scrolling!?

This is useful for long webpages, menus, tutorials, or navagation sections

Code

In this example, Im going to have the same plage navigate to the "More Lessons" section at the bottom of my page.

On href create your id, start with a # and then create an id name like below:

<a href="#navigation">Go to Navigation Section</a>

Then, in the place you want to navigate to, add your id like so. Make sure the names match: 


            <iframe src="navigationPage.html" id="navigation" width="100%" height="250px" frameborder="0"></iframe>

Output

Try it:

Go to Navigation

These are links between the pages of your own website!

Code

In href just add your other HTML page as I did here. You can also do website links

Output

Since the file in the example ends with .html and is from the same folder as your other html webpages, HTML_3Elements.html is an example of an relative link, which is explained in the next subsection below.

Absolute vs Relative Links

Absolute Links

These are href links that show the FULL WEB ADDRESS. This is good for external websites

Code 
<a href="https://www.google.com">Google</a>
Output
Google


Relative Links

Relative Links link to files and pages in your own project

You DO NOT NEED the full website URL unlike with absolute links.

Code 
<a href="HTML_3Elements.html">HTML Elements Lesson</a>
Output
HTML Elements Lesson
