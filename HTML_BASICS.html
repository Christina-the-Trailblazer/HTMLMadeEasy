Main Pieces

  1. <!DOCTYPE HTML> This is a required declaration at the beginning of your HTML code. It tells the webpage to render using the HTML5 specification.

  2. <html> This is the root element of the html document. This tag is the container for all other HTML elements except for the <!DOCTYPE HTML>

  3. <html lang = "en"> This tag lets search engines and screen readers know that this website is in English.
