1. DOCTYPE and 2. HTML

<!DOCTYPE HTML>
This is a required declaration at the beginning of your HTML page. It tells the webpage to render using the HTML5 specification.

<html>
This is the root element of the html document. This tag is the container for all other HTML elements except for the <!DOCTYPE HTML>

</html>
This is the closing tag for the root <html> element.

<html lang = "en">
This tag lets search engines and screen readers know that this website is in English.

ONCE AGAIN <!DOCTYPE HTML> is required for ALL HTML pages.

The HTML document ALWAYS BEGINS with <html> and ALWAYS CLOSES with </html>

3. HEAD TAG

<head>
This tag is the container for the metadata (data about the document). Its placed between the <html> and <body> tags. The content within the <head> is nnot displayed on the webpage. It does provide essential information to the browser and search engine on how to understand the content of the page.

<meta charset="utf-8">
The <meta> tag provides espcific character encoding for the HTML document which ensures that the browser correctly interprets the characters and symbols for the webpage. UTF-8 specifies the character from all languages (Unicode) into bytes for computers.
<title>
The <title> tag defines the webpage's title. This is important for search engine optimization (SEO) - which I will talk about in a later lesson. Its placed inside the <head> element and should only contain text.

DONT FORGET about the CLOSING TAGS: </head> and </title> tags.
