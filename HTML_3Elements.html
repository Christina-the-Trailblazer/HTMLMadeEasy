What are Elements?

An Element is a building block that contains the STRUCTURE and CONTENT of a webpage.

Most elements have:

  • An opening tag
  • Content
  • A closing tag

Examples of an Element:

<p> and </p>
The paragraph element. This is where you would add text to your HTML page.

<body> and </body>
The body element. This is where the content of your webpage goes

<h1> and </h1>
The header element. Dont worry too much on how to use it, as it will be explained in a further lesson.

Void Elements

Void elements are a type of element that only have start tags and contain no content in them

They serve to insert standalone items like images or line breaks

Examples of Void Elements:

<br>
This tag inserts a single line break to your page.

<hr>
This tag creates a thematic line break in a webpage through a solid horizontal line. The purpose is to create visually separating content sections like how you see on this webpage.

<img>
This tag is used to embed a specific imagine in an HTML page. This tag will be explored more in detail in further lessons.
