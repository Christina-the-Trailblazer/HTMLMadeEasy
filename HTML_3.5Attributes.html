Lesson 3.5: HTML Attributes

By the end of this lesson, the learner will Learn:

  • What is an attribute?
  • Why do attributes matter?
  • Basic structure of attributes
  • A few common examples of attributes

What is an Attribute?

An attribute provides extra information to an HTML element.

They are placed inside the opening tag.

Why Do Attributes Matter?

Attributes control how elements behave.

They can define:

  • Destination for links
  • Image sources
  • Descriptions
  • Sizes

Basic Structure of an Attribute

Attributes follow a simple structure made up of a name and a value.

What do each parts of the structure mean?

  1. Attribute Name
    • identifies the attribute
  2. Equals Sign (=)
    • connects the name and value
  3. Value in Quotes
    • provides the information for the attribute

Basically: 

           
               attribute="value"
  • The name (attribute) says WHAT
  • The value says WHICH ONE/WHAT KIND

Examples of Attributes:

  • href
    • Used in links
    • Tells browser where the link goes
  • src
    • Used in images
    • Tells the browser where the image field is located
  • alt
    • Describes the image
    • Helps with accessibility for screen readers
  • width
    • Describes how wide an image should be
  • height
    • Describes how tall an image should be

Examples Using Attributes

NOTICE how ALL the ATTRIBUTES are WRITTEN INSIDE the OPENING TAG using the format name="value"

href

<a href="https://example.com/">Visit Webpage</a>

src

<img src="flower.png">

alt

<img alt="A pretty flower">

width and height

<img width="170" height="170">
