Whats HTML?

HTML stands for Hyper Text Markup Language. Its a scripting language for websites. NOT a programming language!

It determines the structure of the webpage.

HTML also tells browsers how to display the content

HTML Brief History

  • HTML was invented by Tim Berners-Lee

  • Berners-Lee was also credited as the inventor of the World Wide Web (WWW).

  • He creatd HTML to organize and share research documents over the web!

HTML vs. CSS vs. JavaScript

  • HTML (Hyper Text Markup Language) - determines the structure of the webpage, and tells browsers how to display the content.
  • CSS (Cascading Style Sheets) - scripting language that determines the style and presentation of a webpage.
  • JavaScript - programming language that determines the functionality of a website.
