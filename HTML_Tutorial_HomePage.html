What is HTML?

HTML stands for HyperText Markup Language. It's a markup language used to create webpages. HTML is NOT a programming language!

It determines the structure of a webpage.

HTML also tells browsers how to display the content.

HTML History

HTML was invented by Tim Berners-Lee.

Berners-Lee was also credited as the inventor of the World Wide Web (WWW).

He created HTML to organize and share research documents over the web!

HTML was proposed in 1989 and first implemented around 1990–1991 at CERN (European Organization for Nuclear Research).

HTML vs. CSS vs. JavaScript

HTML (HyperText Markup Language) - determines the structure of a webpage.

CSS (Cascading Style Sheets) - determines the style of a webpage.

JavaScript - determines the functionality of a webpage.
