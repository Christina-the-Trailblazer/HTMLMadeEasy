Lesson 2: Basic HTML Structure

By the end of this lesson, the learner will:

  • Recognize the basic structure of an HTML document
  • Identify the purpose of the <!DOCTYPE html>, <html>, <head>, and <body> tags
  • Understand how these elements work together to form a webpage

Hello World Code

<!DOCTYPE html>
<html>
  <head>
    <title>Hello World</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

This is the smallest complete HTML page you can write! Every website starts with this basic structure.

Explanation:

<!DOCTYPE html>
This line must always be the first line of an HTML document. It tells the browser: "This page is written in modern HTML."
<html>
This tag wraps everything on the page. It tells the browser: "Everything inside here is HTML.”
<head>
The <head> element contains information about the page. This information is for browsers and search engines only. You will not see any of this content on your HTML webpage.
<body>
The <body> element contains EVERYTHING that will appear on your webpage.
<h1>
This is a heading tag for your HTML page. More will be explained in later lessons.

Opening and Closing Tags

Most HTML elements use an opening tag and a closing tag.

The opening tag starts the element. The closing tag ends it.

Example

<p>This is a paragraph</p>
  • <p> → opening tag
  • </p> → closing tag (notice the /)
  • Content → goes between the tags

Everything between the opening and closing tags belongs to that element.

How an HTML Page Is Structured

<!DOCTYPE html>         → document type
    <html>              → page wrapper
      <head>            → page information
      <body>            → visible content
    </html>

This structure is used on every HTML page. While the content inside may change, the overall layout always follows this pattern.

Summary

All HTML pages work the same way:

  • The <!DOCTYPE html> declaration tells the browser what type of document it is reading
  • The <head> section stores important information about the page
  • The <body> section displays the content you actually see. Together, these elements work as a team to create a complete webpage.
