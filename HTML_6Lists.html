Lesson 6: HTML Lists

By the end of this lesson, the learner will Learn:

  • What lists are in HTML
  • The difference between ordered and unordered lists
  • How to create lists using the <ul>, <ol>, and <li> tags
  • Creating a terms and definition list with <dl>, <dt>, and <dd>

What are lists?

Lists allow you to organize and group related items

The purpose of using lists in your HTML website is to make content easier to read.

Lists are commonly used for:

  • Steps
  • Menu and Navagation List for your site (like near the footer)
  • Item collections
  • Bullet Points
  • Ranked Lists (like leaderboards)
  • Glossaries and FAQs (Frequently Asked Questions)

Types of Lists and the Differences Between Them

There are two different types of lists:

  • Unordered Lists
  • Ordered Lists

Unordered Lists:

  • Organized in Bullet Points
  • Order Doesnt Matter
  • These are good for non-step information

Examples:

  • Favorite Foods
  • Hobbies
  • Shopping Lists

-----------------------------------------------------------------------------------------

Ordered Lists:

  • Uses Numbers instead of Bullet Points
  • Order MATTERS
  • Good for instructions and ratings

Examples:

  • Recipe Steps
  • Directions
  • Song Rankings

How to Make Lists

Before you choose to make an ordered or unordered list, its important for you to know about <li> or the List Items element

<li> Element

Each list item, no matter ordered or unordered uses these nezted inside

The purpose of the <li> Element is to create individual entries within a list

In the next examples, I will show how <li> is used in ordered and nonordered lists

Ordered Lists

Use the <ol> tag and inside use the <li> tag 


            <p>How to make a cup of tea:</p>
            <ol>
                <li>Boil the water on your stove</li>
                <li>Place a tea bag in a mug</li>
                <li>Pour the hot water</li>
                <li>Let the tea bag simmer for 3 minutes</li>
                <li>Add milk and stir</li>
            </ol>

Result:

How to make a cup of tea:

  1. Boil the water on your stove
  2. Place a tea bag in a mug
  3. Pour the hot water
  4. Let the tea bag simmer for 3 minutes
  5. Add milk and stir

Unordered Lists

Use the <ul> tag and inside use the <li> tag 


            <p>Shopping List:</p>
            <ul>
                <li>Eggs</li>
                <li>Bread</li>
                <li>Milk</li>
                <li>Tea</li>
                <li>Cheese</li>
            </ul>

Result:

Shopping List:

  • Eggs
  • Bread
  • Milk
  • Tea
  • Cheese

BONUS: Making a Terms and Definition List

Now that you learned the basics on lists, you can also use lists to make terms and definitions Known as a Description List

There are three Tags:

  • <dl> creates the description list
  • <dt> defines the term
  • <dd> defines the description or definition

    • Description List Example

    
        <dl>
            <dt>HTML</dt>
            <dd>Used to structure webpages</dd>

            <dt>CSS</dt>
            <dd>Used to style webpages</dd>

            <dt>JavaScript</dt>
            <dd>Used to add interactivity to webpages</dd>
        </dl>

    RESULT:

    HTML
    Used to structure webpages
    CSS
    Used to style webpages
    JavaScript
    Used to add interactivity to webpages
